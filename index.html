The big idea
🤔

What is vibe coding?

Vibe coding is a new way of building software where you describe what you want in plain English, and an AI assistant writes the actual code for you. You don't need to memorize programming languages, syntax rules, or technical jargon — just explain what you're trying to build.

Think of it like this: instead of learning to drive a race car, you tell a world-class driver exactly where you want to go. The AI handles the technical driving — you handle the vision and direction.

The tools you need

These free tools work together to take your idea from a blank screen to a real, live website. Add Supabase when your project needs to store data.

Claude Code

Your AI coding partner
Lives right inside VS Code. Type what you want in plain English and it writes the code, edits your files, and explains everything — no terminal needed.

A chat window where you describe what to build and it builds it

VS Code

Your home base
A free app from Microsoft. Your files on the left, code in the center, Claude Code chat on the right. Everything in one window — this is where you'll live.
Think of it as
The one app you have open the whole time
Get it free →
🐙

GitHub

Your backup & history
Saves every version of your project online. If something breaks, you can always roll back. Used by everyone from solo hobbyists to major tech companies.
Think of it as
A save point system — always there if you need to go back
Sign up free →
▲

Vercel

Your publisher
Takes your code from GitHub and puts it live on the internet — automatically. Every time you save a change, your live site updates. No extra steps, ever.
Think of it as
The 'publish' button for the entire internet
Sign up free →
🗄️

Supabase

Your database
A free, open-source backend that gives your app a real database, user authentication, and file storage — all without writing server code. Perfect when your project needs to save and retrieve data.
Think of it as
A spreadsheet in the cloud that your app can read and write to
