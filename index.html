The big idea
🤔

What is vibe coding?

Vibe coding is a new way of building software where you describe what you want in plain English, and an AI assistant writes the actual code for you. You don't need to memorize programming languages, syntax rules, or technical jargon — just explain what you're trying to build.

Think of it like this: instead of learning to drive a race car, you tell a world-class driver exactly where you want to go. The AI handles the technical driving — you handle the vision and direction.

Best of all, this approach works with many different AI models and tool combinations. This guide is tool-agnostic — the concepts apply regardless of which coding assistant or editor you choose.

Your toolkit
The building blocks you need

Vibe coding brings together five categories of tools — each playing a different role. Many options exist for each category, so you can mix and match what works for you. The examples listed are popular choices, not the only ones.

🤖

AI Coding Assistant

The brain of your setup
Understands plain English and writes code for you. Ask it to build features, fix errors, or explain what the code does — no programming knowledge required. Different models have different strengths, so it's worth trying a few.
Popular examples
Claude Code, GitHub Copilot, ChatGPT / Codex, Gemini Code Assist, and more
💻

Code Editor

Your home base
The app where you manage your project files and chat with your AI assistant — usually all in one window. Some editors have AI built right in; others let you add it as a plugin or extension.
Popular examples
VS Code, Cursor, Windsurf, Conductor, Replit (browser-based), Zed, and more
🐙

Version Control

Your backup & history
Saves every version of your project online. If something breaks, you can always roll back. Also how your code connects to your deployment platform so updates go live automatically.
Popular examples
GitHub, GitLab, Bitbucket
▲

Deployment

Your publisher
Takes your code and puts it live on the internet — automatically. Every time you push a change to version control, your live site updates with no extra steps needed.
Popular examples
Vercel, Netlify, Railway, Render, GitHub Pages
🗄️

Database & Backend

Your data layer (optional)
Adds a real database to your app so it can store and retrieve data — user accounts, form submissions, lists, and more. You don't need to write server code; describe what you want to store and your AI assistant handles the rest.
Popular examples
Supabase, Firebase, Neon, PlanetScale — add this when your project needs to save data
